# Unsupervised-Learning-Clustering

# Unsupervised Learning: Clustering Techniques

## Overview
This project demonstrates the implementation and comparison of three popular unsupervised machine learning algorithms: K-Means Clustering, Hierarchical Clustering, and DBSCAN. The project includes data preprocessing, exploratory data analysis (EDA), feature scaling, clustering, model evaluation, and visualization.

## Dataset
**Dataset:** Mall Customer Segmentation Dataset

The dataset contains customer information such as Age, Annual Income, and Spending Score, which are used to identify customer segments.

## Algorithms Implemented
- K-Means Clustering
- Hierarchical Clustering (Agglomerative)
- DBSCAN

## Evaluation Metrics
- Silhouette Score
- Davies-Bouldin Index

## Visualizations
- Exploratory Data Analysis (EDA)
- Elbow Method
- Dendrogram
- Cluster Scatter Plots

## How to Run
1. Clone the repository.
2. Install the required dependencies.
3. Open the notebook in the `notebooks` folder.
4. Run all cells to reproduce the clustering analysis.

## Results
- K-Means achieved the highest clustering quality based on Silhouette Score and Davies-Bouldin Index.
- Hierarchical Clustering produced similar results with effective cluster visualization using a dendrogram.
- DBSCAN successfully identified clusters and demonstrated density-based clustering, though its performance depended on parameter selection.
